CHARACTERS

Character Creation

Character creation works as follows:

  • Select character class for PC(s), up to point maximum
  • For each PC, roll stats ATK, AIM, IQ, etc.
  • For each PC, from class features, make note of #attk. #def, etc.
  • If points remain, select additional PC(s) as above.

Altneratively, players could be allowed to roll stats first, then select a class.

The meaning of these terms will be explained in what follows.

Stats and the Dice Chain

ATK, AIM, DEF, IQ, etc.

In most RPGs, a character has a certain set of "stats", usually in the range 3-18 by rolling 3d6 in order. These stats then correspond to bonuses from -3 to +3.

In Brigandine your "stat" is a die, usually a d6, and is generated by rolling 4d2. The stat is the die that has that many sides. Any time the game requires rolling dice to resolve a situation, you will roll the dice corresponding to one of your stats. The principal stats are:

ATK
Attack, representing physical ability and prowess. Almost always the die rolled in melee combat.>/dd>
AIM
Aim, representing the ability to target individuals. This includes things like good eyesight and hand-eye coordination.
DEF
Defense, representing ability to avoid damage.
IQ
Intelligence, representing mental ability and ease of solving technical problems.

The Dice Chain

When bonuses or penalties are to be applied, they should be applied to the stat to be rolled according to the dice chain:

d0-d1-d2-d3-d4-d5-d6-d7-d8-d10-d12-d14-d16-d20-d24-d30

A +1 bonus means move up one step in the dice chain. For example, if the usual die is a d6, to apply a +1 bonus, instead roll a d7. To apply a -1 penalty to a d6, instead roll a d5. If the usual die is a d8 and there is a +3 bonus, instead roll a d14.

The standard base die is the d6, which represents the ability of an average man. Many rolls exploring the dungeon will be opposed rolls against a target die, also usually chosen to be a d6.

The weird ones

The dice d0,d1,d2,d3,d5,d7,d14,d16,d24,d30 are not as well beloved as their Platonic solid cousins. They can nevertheless be found for sale online from dice shops or other RPG supplies.

d0, d1
These are simply constants, either 0 or 1. A d0 is automatic failure.
d2
Most familiarly, a coin flip. Can also be represented using a cubical die ("d6"), by writing the numbers "1" and "2" on the faces, three times each. Even better, color one side green, the other red, and treat "green=2", "red=1".
d3
There are some specially shaped d3. Can be more simply represented using a cubical die ("d6"), by writing numbers "1"-"3" on the faces, twice each. Color three sides green, three red for a dual d2/d3.
d5
Specially-shaped d5s that look like triangular prisms. Also possible by double-labeling a d10 (treat 6-10 as 1-5).

Exploding Dice

When rolling a d6, any natural roll of 6 is considered to "explode", and is then re-rolled. If the result is again 6, then the result is considered a 7 and re-rolled. If again a 6, then considered an 8 and re-rolled. This continues until a number other than 6 is rolled. In this way, a d6 is still able to reach a result above the usual range.

This applies not only to d6, but any die in the dice chain, on a maximum roll (i.e. on a natural X on a dX).

For instance, a result of 5 on a d5 is re-rolled, and if 5 rolled again it is considered a 6 and re-rolled, and if 5 again considered a seven, etc.

hits, #attk, #def, attk type, def type, morale

As in most other RPGs, the principal stats are the not the other numerical categorizations of a PC. There are other factors which determine the ability of a PC, in particular in combat.

hits
A measurement of the toughness of the character, in terms of the number of times the character may survive being struck by an otherwise lethal blow. For normal people, this is always 1. The value is determined by class and level.
#attk
For a powerful warrios who fights with the strength of several men; this characterizes exactly how many men he fights with the strength of. If 1, fight as single man; if 2, fight as 2 men; etc. Is necessary for determining how many dice to roll on skirmish combat. The value is determined by class and level.
#def
As above, but represents how well a warrior can defend himself in battle. Is also necessary for determining how many dice to roll on skirmish combat. The value is determined by class and level.
type attk
The type of soldier that the character is classified as in melee attacks: LF, MF, HF, LC, MC, HC. This is also necessary for determining how many dice to roll on skirmish combat. The value is determined by class and equipped weapon.
type def
The type of soldier that the character is classified as in melee defense: LF, MF, HF, LC, MC, HC. This is also necessary for determining how many dice to roll on skirmish combat. The value is determined by class and worn armor class.
morale
The willingness and resolve of the character to fight on. On a morale check, a character must score under this score on 2d6 to continue fighting. Rarely checked for PCs. Determined by class and level.

In any combat, it is necessary to know:

  • type and # attk
  • type and # def

which are all determined by

Fantasy Combat Type (FC)

Some creatures are exceptionally powerful, and can only be vanquished using the Fantasy Combat rules. Certain character classes are able to advance onto the Fantasy Combat tables, and if so are given a FC type. This only occurs for certain classes and at higher levels.

Point Values

All characters in the game have an assigned point-value. This originates from the war game roots. To create balanced armies in a war game, each side would be assigned a certain number of points, and each troop type a certain point value, and opponents would construct their armies within the point budgets.

For Brigandine, each player has a point budget of 5.

Players are allowed to control more than one character, provided the total value of all PCs does not exceed 5.

Whenever a player loses a PC, the player regains those points, which can be used at a later (reasonable) time to attempt to recruit additional characters, assuming the player's point value remains below 5.

Players may also swap PCs by deliberately dropping one, thereby regaining the points, then picking up the other, expending points. The total point value for the player must remain below 5. PCs swapped out in this way do not continue to tag along with the party; anyone coming along must be controlled by a player. Any PC not controlled will immediately return to civilization and remain there conducting business relevant to himself.
