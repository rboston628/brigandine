OPTIONAL CLASSES

Optional Classes

The main classes draw directly from soldier types in the medieval wargame rules. They are inspired by cavalry, infantry, arquebusiers, archers, and other troop types, and implement when possible some of the special rules for these troops as class features.

These optional classes are obliquely related to medieval wargame rules. Most of them, such as the demihumans and mage, come from the fantasy supplement. Others are suggested by time period, alternative medieval wargame rules, and common RPG conventions.

Below are some optional classes, along with their point-buy values. Whether any of these is used in the game is entirely the decision of the referee. Each will be explained separately.
