OPTIONAL CLASSES

Optional Classes

The main classes draw directly from soldier types in the medieval wargame rules. They are inspired by cavalry, infantry, arquibusiers, archers, and other troop types, and implement when possible some of the special rules for these troops as class features.

These optional classes are obliquely related to medieval wargame rules. Some, such as the demihumans and mage, come from the fantasy supplement. Others are suggested by time period, alternate medieval wargame rules, and common RPG conventions.

Below are some optional classes, along with their point-buy values. Whether any of these is used in the game is entirely your (the referee's) decision. It is recommended, if they are introduced at all, they are only introduced after play and exploration. Each will be explained separately.

Class Features

Magic
A knight need not check morale in combat.

Source

Though canonized with the "standard four," the thief class was a late arrival to RPGs, originating as a homebrew class published as a user submission to an RPG magazine. The original was based on the stories of Fafhrd and the Grey Mouser.

Class Features

Miracles
A cleric may be able to perform miracles.

Starting Equipment

holy symbol, robe, pyx box, prayer book.

Source

Though the cleric class pre-dates the official publication of RPG rules, it also has no precedent in medieval wargames.

The closest is a late edition of the WRG Ancient rules, which includes religious figures as an optional soldier type. However, these religious figures only serve the role of a morale booster, similar to a banner. The effect is purely due to the fear the soldiers have of the priests' deities.

The original RPG cleric was properly a vampire hunter based on VanHelsing from Hammer horror films, and not based on any real religious figure of legend. It otherwise doesn't make sense to introduce a priest as a dungeon-delving class.

Class Features

Magic
A mage can use magic and cast deadly spells. A spell is treated as artillery. It must be declared at the start of a combat round. It takes the mage time to properly focus, pronounce the words, and perform the somatic movements. The spell does not go into effect until the artillery phase. If the mage is interrupted by melee before the artillery phase, then the spell is lost.

Starting Equipment

staff, robe, mostly blank spellbook.

Source

The fantasy supplement to the medieval wargame rules describes a wizard, which is entirely over-powered to fit into Brigandine. This mage has been significantly pared-down.
